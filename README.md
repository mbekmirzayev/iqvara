Iqvara

Iqvara — nomi Iq (Iqbol) va vara (“yangi era”) birikmasidan olingan. Bu loyiha do‘stlikdan ilhomlangan bo‘lib, mening birinchi onlayn kurs platformam sifatida yaratilmoqda.

About

Iqvara — online kurslar joylash uchun mo‘ljallangan web-platforma. Maqsad: minimal, tushunarli va kengaytiriladigan o‘quv tizimi yaratish.

Features

* Courses & Sections (CRUD)

* Lessons with video or document links

* User roles: Admin, Instructor, Student

* Authentication & Registration

* Simple admin management

Tech Stack

* Backend: Django / Django REST Framework

* Frontend: (React, Vue yoki boshqa SPA — alohida loyiha sifatida)

* Database: PostgreSQL

* Optional: Redis (caching)
